## Installation and Setup Instructions

Clone down this repository. You will need `npm` and `node` installed on your machine.  

Installation:

`npm install` 

`npm install react-dom`

`npm install vite`

`npm install react-beautiful-dnd`

To Run App Locally:  

`npm run dev`

`http://localhost:5173/`
